# AAC Speech & Communication Website

Static one-page GitHub Pages site for the AAC Speech & Communication app.

## Local development

1. Install Node.js if you do not already have it.
1. Run `npm run dev`.
1. Open `http://127.0.0.1:4000/` in your browser.
1. The dev server reads the same Jekyll-style files, so the GitHub Pages version stays unchanged.

## What's included

- Hero section with Google Play CTA
- Feature blocks for free and premium tiers
- Screenshot gallery using responsive SVG mockups
- SEO metadata, Open Graph tags, JSON-LD, `robots.txt`, and `sitemap.xml`
- Local preview server that renders the same templates without changing GitHub Pages output

## Update the store link

The Google Play button uses the store URL in `google_play_url` inside [`_config.yml`](_config.yml).
